# ruuvitag-data
Save RUUVITAG_MAC environment variable before running.

``` export RUUVITAG_MAC="12:34:56:AB:CD:EF" ``` (replace placeholder with correct MAC)

| Flag | Description |
| --- | ----------- |
| -v / --verbose | Enables verbose mode (log.info) |
| -vvvv / --superverbosemode | Enables debugging mode (this WILL spam your console) |
